# ChromeDTP
A [Chrome DevTools Protocol](https://chromedevtools.github.io/devtools-protocol/) utility library

#### Languages
* [Java](https://en.wikipedia.org/wiki/Java)

#### Tools and libraries
* [Chrome DevTools Protocol](https://chromedevtools.github.io/devtools-protocol/) - For browser control/embedding in application
* [MinGW](https://www.mingw-w64.org/), [GCC](https://gcc.gnu.org/) and [JNI](https://en.wikipedia.org/wiki/Java_Native_Interface) - For compiling utility sub-library _windowHelper_
